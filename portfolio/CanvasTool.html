Canvas Notifyer

Aug--Nov 2023

❮ ❯

Canvas Notifyer is a GUI Python program that retrieve and store data from Canvas (course management system used in UBC). It shows the assignments that due within a week on the homepage, and has separate pages for assignments and announcements. In assignment page, it will show each assignment's name, due date, points and score received in a table format, and supports the feature of checking description for specific assignment


I personally:

· Integrated Canvas API to fetch real-time content

· Implemented a database system using SQLite to ensure off-line access
