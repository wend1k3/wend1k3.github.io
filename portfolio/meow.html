Meow May Cry

June 2024

❮ ❯

Meow May Cry is a 2D two-player fighting game in which the black cat character seeks revenge to the church and fight against the pope character. The black cat character can move, jump, block and has melee attack, while the pope character has two full-range attacks and a special skill for disappearing


I personally:

· Programmed player controller, including movement and combat mechanics

· Implemented animations for each skill

· Implemented UI for health system and stamina bar of blocking


This is my first time doing a game jam and I found it quite different from hackathons. Due to time conflict, we only had three days to build this game. Thankfully we managed to pull it off, thanks to effective communication. As for improvement plan, we plan to modify the pope character into a AI enemy, like those common fighting games.
